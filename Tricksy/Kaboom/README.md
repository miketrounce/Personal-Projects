# Kaboom

Live page:

[https://miketrounce.github.io/Tricksy/Kaboom/](https://miketrounce.github.io/Tricksy/Kaboom/)

A playful chemistry-chaos game where you mix suspicious ingredients, chase
ridiculous round objectives, and try not to destroy the lab before the run is
over.

## Run locally

From the repo root:

```bash
npm start
```

Then open [http://localhost:4173/Tricksy/Kaboom/](http://localhost:4173/Tricksy/Kaboom/).

## Goal

Pick two chemicals, set the heat level, and make the funniest possible mess
across six rounds.

- Match the round objective when you can
- Rack up chaos points
- Try not to let lab safety collapse to zero

## Systems

- Six chemistry ingredients with different reaction personalities
- Deterministic reaction resolution for testing
- Round-by-round objectives
- Heat level control
- Score, safety, streak, and reaction log tracking
