# Personal Portfolio

Live page:

[https://miketrounce.github.io/Personal-Projects/portfolio/](https://miketrounce.github.io/Personal-Projects/portfolio/)

A simple personal portfolio starter page inside the shared `Personal-Projects` repo.

## Run locally

From the repo root:

```bash
npm start
```

Then open [http://localhost:4173/portfolio/](http://localhost:4173/portfolio/).

## What to customize

- Name and intro copy
- About section content
- Real project list and links
- Contact links
